# 测试命令

你好！如果你能看到这个响应，说明命令系统工作正常。

## 可用的命令

- `/workflow <功能名称>` - 完整工作流
- `/stage-1-user-story <功能名称>` - 创建用户故事
- `/stage-2-bdd-scenario <功能名称>` - 设计 BDD 场景
- `/stage-3-tdd-cycle <功能名称>` - TDD 开发循环
- `/stage-4-implementation <功能名称>` - 实现代码
- `/stage-5-optimization <功能名称>` - 代码优化
- `/tdd-quick <功能名称>` - 快速 TDD 工作流

## 示例

```
/workflow 用户登录
```
