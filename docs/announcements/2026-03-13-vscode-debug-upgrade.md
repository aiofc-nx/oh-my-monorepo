## 🚀 如何使用

### 快速开始（30秒上手）

1. **打开项目**

   ```bash
   code /path/to/oh-my-monorepo
   ```

2. **按 F5 启动调试**
   - VSCode 会显示配置选择菜单

3. **选择配置类型**
   - `Debug App (NestJS)` - 后端应用
   - `Debug App (React + Vite)` - 前端应用
   - `Debug Jest Tests` - 单元测试
   - 等等...

4. **输入项目名称**
   - 从下拉列表中选择（如 `api`, `web`）

5. **设置断点并开始调试**
   - 在代码行号左侧点击设置断点
   - 按 F5 开始调试

## ❓ 常见问题

### Q1: 找不到项目选择菜单？

**A**: 确保你按了 F5，而不是其他快捷键。如果问题依旧，尝试：

1. 重启 VSCode
2. 检查 `.vscode/launch.json` 文件是否正确

### Q2: 断点不生效？

**A**: 检查以下几点：

1. source maps 是否启用（配置中已启用）
2. 端口是否被占用
3. 项目是否正确编译

### Q3: 如何添加新项目？

**A**: 无需修改配置！新项目会自动出现在选择列表中。只需在 `inputs` 部分的 `options` 数组中添加项目名称。
