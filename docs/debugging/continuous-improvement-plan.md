## 🔧 快速修复指南

### 常见问题和解决方案

#### 问题 1: 配置不显示

**症状**: 按 F5 后看不到新配置

**解决方案**:

```bash
# 1. 重载 VSCode 窗口
Cmd/Ctrl + Shift + P → "Reload Window"

# 2. 检查配置文件
cat .vscode/launch.json | jq '.configurations[].name'

# 3. 清除 VSCode 缓存
rm -rf ~/.vscode/User/workspaceStorage
```

#### 问题 2: 断点不生效

**症状**: 断点显示灰色，代码不暂停

**解决方案**:

```bash
# 1. 检查 source maps
pnpm nx build <project> --sourceMap

# 2. 验证 outFiles 路径
ls -la apps/<project>/dist/

# 3. 检查 tsconfig
cat apps/<project>/tsconfig.app.json | grep sourceMap
```

#### 问题 3: 端口被占用

**症状**: 启动失败，提示端口已被使用

**解决方案**:

```bash
# 1. 查找占用进程
lsof -i :9229

# 2. 终止进程
kill -9 <PID>

# 3. 或使用其他端口
NODE_OPTIONS="--inspect=9230" pnpm nx serve <project>
```
